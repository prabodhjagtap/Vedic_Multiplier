For implementation of Vedic square multiplier both half and full adder modules are required.
For implementation of 2 bit square circuit using Vedic multiplier method, 2 half adder modules are required and 4 AND gate primitives are required.
For implementation of 4 bit square circuit using Urdhwa-Tiryabhyam Sutra, half adder module, Full adder module and AND gate primitives are required.
